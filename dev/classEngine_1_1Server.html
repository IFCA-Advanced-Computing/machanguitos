void Engine::Server::addAgents ( const std::string &  name,
const unsigned  n 
)

Add agents number to create them in the future.

This method don't create the agent rigth now. It only adds to an internal list. It's possible to execute several times with the same name. E.g:

Server::instance()->addAgents( "cow", 10 );
Server::instance()->addAgents( "cow", 10 );

Has the same result of:

Server::instance()->addAgents( "cow", 20 );
Parameters
nameof the AgentClass.
nnumber of Agents to create.

void Engine::Server::setRasterLayerUpdate ( const std::string &  key,
const std::string &  file 
)

Set a Lua update source for a Raster object.

The Lua file should implement a 'Raster:update' function like:

function Raster:update( delta )
io.write( "Fire update " .. delta .. "\n" )
for j = 0, self.height - 2 do
for i = 0, self.width - 1 do
local p0 = 0
if i > 0 then
p0 = self:getpixel( 0, i-1, j+1 )
end
local p1 = 0
if i < self.width - 1 then
p1 = self:getpixel( 0, i+1, j+1 )
end
local p2 = self:getpixel( 0, i, j+1 )
local p3 = self:getpixel( 0, i, j )
local newval = (p0 + p1 + p2 + p3) / 4.0
self:setpixel( 0, i, j, newval )
end
end
end
Parameters
keyname of the Raster object.
filelua filename.
