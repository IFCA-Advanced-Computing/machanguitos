void Engine::Server::addAgents ( const std::string &  name,
const unsigned  n 
)

Add agents number to create them in the future.

This method don't create the agent rigth now. It only adds to an internal list. It's possible to execute several times with the same name. E.g:

Server::instance()->addAgents( "cow", 10 );
Server::instance()->addAgents( "cow", 10 );

Has the same result of:

Server::instance()->addAgents( "cow", 20 );
Parameters
nameof the AgentClass.
nnumber of Agents to create.
