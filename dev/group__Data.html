int Data::data_createLayer ( lua_State *  L)

Define data.createLayer library function.

This function create a layer in the datastore.

data.createLayer( "grass",
{
x0 = 0.0,
x1 = 1.0,
y0 = 0.0,
y1 = 1.0,
w = 100,
h = 100,
default = 250,
} );

It create a layer with 1 sublayers. But it is possible to specify more sublayer with the parameter `layers`, like:

data.createLayer( "grass",
{
x0 = 0.0,
x1 = 1.0,
y0 = 0.0,
y1 = 1.0,
w = 100,
h = 100,
default = 250,
layers = 3,
} );

The layers are created with Byte type (range [0..255]) but it is possible to ask to use Float number (32 bits Float) in the layers using the parameter `isFloat`, like:

data.createLayer( "grass",
{
x0 = 0.0,
x1 = 1.0,
y0 = 0.0,
y1 = 1.0,
w = 100,
h = 100,
default = 250,
isFloat = true,
} );

Float raster layers will be defaulting to byte when saved on formats that don't allow Float types (like png, or jpeg). You can use TIFF files to save them with Float numbers.
